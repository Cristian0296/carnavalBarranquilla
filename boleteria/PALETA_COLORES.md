# Brief de Rediseño Visual

## Objetivo

Rediseñar visualmente la página web para alinearla con la identidad gráfica del logo del Festival de Barranquilla en Atlanta.

La experiencia debe sentirse:

- Festiva
- Tropical
- Cultural
- Vibrante
- Moderna

El resultado debe mantener legibilidad, buena experiencia de usuario y una apariencia profesional, pero evitando una estética sobria, bancaria, gris o demasiado neutra.

## Inspiración

La dirección visual debe inspirarse en:

- Carnaval de Barranquilla
- Fiesta
- Cultura caribeña
- Color
- Energía
- Música
- Tradición

## Paleta de Colores

- Azul oscuro principal: `#0D2A5C`
- Azul medio: `#1F5FBF`
- Amarillo dorado: `#F5C518`
- Rojo vibrante: `#E53935`
- Naranja: `#FF7A00`
- Verde acento: `#2ECC71`
- Azul claro acento: `#00AEEF`
- Blanco: `#FFFFFF`

## Aplicación de Colores

- Reemplazar el fondo negro por azul oscuro `#0D2A5C` o un degradado entre `#0D2A5C` y `#1F5FBF`.
- Eliminar bloques grises planos y reemplazarlos por fondos claros cálidos o tarjetas con mejor contraste y estilo más atractivo.
- Usar el rojo `#E53935` para botones principales.
- Usar el amarillo `#F5C518` para highlights, hover states, detalles y títulos destacados.
- Usar naranja, verde y azul claro como colores de apoyo en detalles visuales, iconos, bordes o acentos.
- Mantener buen contraste y accesibilidad.

## Cambios de Diseño Requeridos

### 1. Header / Navbar

- Hacerlo más visualmente atractivo.
- Usar fondo azul oscuro o degradado azul.
- Mejorar botones o enlaces del menú con hover colorido.
- Integrar mejor el logo visualmente con el header.
- Agregar una separación visual limpia, moderna y alineada con el branding.

### 2. Hero Section

- Rediseñar completamente el bloque principal para que se vea más dinámico y festivo.
- Quitar la apariencia de caja gris plana.
- Usar un fondo más atractivo, con color, profundidad o degradados suaves.
- Mantener jerarquía visual clara.
- Hacer que el título principal destaque mucho más.
- Mantener el subtítulo legible y complementario.
- Dar a los botones CTA una apariencia moderna, redondeada y alineada con los colores del festival.

### 3. Botones

- "Iniciar sesión" debe ser el botón principal con fondo rojo `#E53935` y texto blanco.
- "Registrarse" y "Ver disponibles" deben funcionar como botones secundarios alineados al branding.
- Agregar hover states modernos y elegantes.
- Mejorar bordes, radios y padding.

### 4. Tipografía

- Mantener una fuente limpia y moderna.
- Dar más carácter visual a los títulos.
- Mejorar tamaños, pesos y espaciados.
- El título principal debe sentirse más enérgico y alineado con el concepto del festival.

### 5. Estilo General

- Añadir de forma sutil elementos decorativos inspirados en carnaval.
- Usar curvas, acentos coloridos y energía visual sin saturar la interfaz.
- Mantener el diseño limpio, usable y responsive.
- Evitar recargar con demasiados elementos decorativos.
- Buscar un resultado premium, alegre y temático.

### 6. Responsive

- Asegurar buena visualización en desktop, tablet y móvil.
- Mantener buena distribución del contenido.
- Escalar correctamente el logo y los botones.

## Instrucciones Técnicas

- No cambiar la estructura funcional de la página si no es necesario.
- Priorizar estilos, colores, espaciados, tipografía y componentes visuales.
- Reorganizar ligeramente el layout solo si mejora el balance visual.
- Si el proyecto usa CSS, actualizar CSS.
- Si usa Tailwind, aplicar los cambios con clases apropiadas.
- Si usa React o componentes, respetar la estructura existente.

## Resultado Esperado

Una página que represente visualmente el Festival de Barranquilla en Atlanta, conectada con el logo y su identidad gráfica, con una estética vibrante, cultural y moderna, usando la paleta indicada y mejorando claramente la experiencia visual.

## Plan de Trabajo por Pasos

1. Actualizar la base visual global: fondo general, variables de color, tipografía y tono visual.
2. Rediseñar el header y la navegación.
3. Rediseñar el hero principal.
4. Ajustar botones y estados interactivos.
5. Refinar tarjetas, bloques de contenido y secciones secundarias.
6. Pulir responsive y consistencia visual general.
